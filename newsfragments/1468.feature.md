Changed `std::evm::crypto` precompile wrappers to return `Result<PrecompileError, T>` instead of reverting on precompile call failure, so verifier and signature-checking code can handle invalid precompile inputs explicitly. Added typed wrappers for the remaining EVM crypto precompiles, including RIPEMD-160, identity, Blake2F, KZG point evaluation, BLS12-381 operations, and P-256 verification.
